# 🛡️ SOC VirusTotal Analysis Tool

A Python-based CLI utility designed for Security Operations Center (SOC) analysts to streamline threat intelligence gathering. This tool interfaces with the **VirusTotal v3 API** to provide rapid verdicts on suspicious artifacts.



## 🚀 Features

* **File Scanning:** Upload local files to VirusTotal and wait for a full analysis report.
* **IP Reputation:** Instantly check if an IP address has been flagged by security vendors.
* **Hash Lookup:** Query SHA-256, SHA-1, or MD5 hashes to see historical scan results without re-uploading the file.
* **Domain Analysis:** Cleanse and check domains/URLs for phishing or malware associations.
